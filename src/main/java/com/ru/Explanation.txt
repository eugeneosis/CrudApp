

                                      MVC - Model View Controller
                                              |     |       |

Controller'ом - Являются методы сервлета Request и Response. Request запрашивает у сервера информацию, Response отдает
Т.е управление запросами пользователя. Это запросы Http get и post.
Model - Являются ДАННЫЕ. Массив или база данных, ключ здесь какие то данные.
View - Представление этих данных через JSP. В нем можно делать какой-то фронт, а так же использовать джававские методы
